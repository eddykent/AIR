### Selenium:
---------
Add chrome.exe to your path to ensure that selenium can run (and that webdriver-manager can do its thing) 
If selenium does not run, it can be caused by either the wrong chrome driver version or by the wrong chrome version

### python:
-------
Use python 3.7 and above
All modules need to be installed from dependencies.txt
